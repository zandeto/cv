# Alexander Petkov — CV

Interactive CV hosted at **https://zandeto.github.io/cv/**

Single-file HTML CV with WebGL particle system, scroll-driven transitions, and
print-optimized PDF export.

## Local preview

Open `index.html` in a browser — no build step, no dependencies.

## Print / PDF

Use the "Print / PDF" button in the top-right, or the browser's native print
dialog. A dedicated print stylesheet renders a single-page ATS-friendly layout.
